Silicon Architecture

Venice SP7 Block Diagram

The Venice SP7 platform is built around AMD's latest Zen6/Zen6c cores with a new 4th-generation Infinity Fabric, PCIe Gen6 I/O, and a massively expanded 16-channel DDR5/MRDIMM memory subsystem.

Memory Subsystem

DDR5 & MRDIMM Support

Venice SP7 delivers a massive leap with 16 memory channels — up from 12 in Turin/SP5 — supporting both DDR5 at 8000 MT/s and the next-generation MRDIMM format at 12.8 GT/s, enabling unprecedented memory bandwidth.

🧠

DDR5 Channels & Speed

  • 16 channels DDR5 with ECC (vs 12 on SP5)
  • RDIMM & 3DS RDIMM: up to 8000 MT/s (1DPC)
  • MRDIMM & Tall MRDIMM: up to 12,800 MT/s (1DPC)
  • Supports 2DPC (DDR5 RDIMM)
  • 16Gb, 24Gb, and 32Gb DRAM die densities
⚡

Memory Interleaving & DIMM Types

  • Support for 1, 4, 8, 12, 16-channel interleaving
  • RDIMM · 3DS RDIMM · MRDIMM · Tall MRDIMM
  • NPS1, NPS2 (performance-focus), NPS4 OPN-specific
  • Memory self-healing (PMU MBIST / MHB)
  • Post Package Repair (PPR) — in-band & out-of-band

Theoretical Peak Bandwidth Comparison

Venice SP7 — MRDIMM 12.8 GT/s × 16ch~3,276 GB/s
Venice SP7 — DDR5 8000 MT/s × 16ch~2,048 GB/s
Turin SP5 — DDR5 6400 MT/s × 12ch~1,229 GB/s

Approximate theoretical peak. Actual bandwidth depends on configuration, frequency grade, and workload.

I/O & Connectivity

PCIe Gen6 · CXL 3.1 · xGMI Gen4

Venice SP7 introduces PCIe Gen6 (64 Gbps/lane) and CXL 3.1 — a significant leap from Gen5/CXL 2.0 in Turin/SP5, with more flexible lane configurations and up to 3 xGMI links for 2P systems.

🔌

PCIe Gen6

64 Gbps/lane — 2× bandwidth vs Gen5. Supports GPU, NIC, Switch, RAID, SSD accelerators.

🔗

CXL 3.1

12.8 GT/s · Type 1/2/3 devices · Memory pooling, tiering, expansion. Upgraded from CXL 2.0.

↔️

xGMI Gen4

3 links at 64 Gbps for 2P. 4th-gen Infinity Fabric. Dynamic link width management.

🖥️

Accelerator I/O

SDXI (Smart Data Accelerator Interface) with Datamover Encryption + SDCI (Smart Data Cache Injection).

Security & AI Inferencing

Hardened Security · AI-Ready

Venice SP7 adds triple-engine security with enhanced Root of Trust, Post-Quantum Cryptography support, and FIPS 140-3 certification. FP16/AVX-VNNI-INT8 ISA updates enable efficient AI inference.

🔒

Triple Security Engines

  • ART (AMD Root of Trust) · Caliptra · ASP (AMD Security Processor)
  • Enhanced RoT with RSA 4K support
  • Post-Quantum Cryptography: PQC (LMS) NEW
  • FIPS 140-3 Level 1 Certification for RoT engines
  • Confidential Compute: SEV, SEV-ES, SEV-SNP, SEV-TIO
  • Platform Secure Boot · Secured Update · Debug · Recovery
  • Platform Attestation · Device Identity
  • Discrete TPM / DRTM support
  • Physical & Side Channel Attack Mitigations
🤖

AI Inferencing Capabilities

  • ISA: FP16 — half-precision floating point NEW
  • ISA: AVX-VNNI-INT8 — vector neural network INT8
  • ISA: FRED — Flexible Return and Event Delivery NEW
  • ISA: PML — Page Modification Logging NEW
  • ISA: IBS-Lite — Memory profiler / page migration
  • ZenDNN framework support
  • High Frequency (HF) SKUs for GPU efficiency
  • Companion to AMD GPUs (MI300X, MI325X, MI355X)
⚠️

RAS & Manageability (MP-RAS)

  • Dedicated MP-RAS sideband processor NEW
  • Platform-First Error Handling (PFEH)
  • OOB Error Management via PLDM/RDE
  • PMIC error reporting (RDIMM, MRDIMM)
  • CXL protocol error handling
  • MPRAS CXL component error handling
  • In-band MCA error reporting
  • Out-of-band: APML SB-RMI / SB-TSI

Development Timeline

Venice SP7 Roadmap & Milestones

From the first engineering samples in August 2025 to production ramp in H2 2026, the Venice SP7 program is on track with regular AGESA PI releases.

ES1
August 2025 · Completed

Engineering Samples 1 — BIOS Bring-Up

48C / 64C / 96C / 128C (CCD32) · Fused non-secure · Intended for BIOS development, bring-up testing, early system validation, electrical margining per guidance.

ES2
November 2025 · Completed

Engineering Samples 2 — System Validation

48C / 64C / 96C / 128C / 144C / 192C (CCD32) · Fused non-secure · AGESA PI 0.0.7.0 (Oct 2025). System stress and validation, electrical compliance and margining.

NOW
Apr 2026 · Current

PI 0.0.7.4 Released — Final Pre-PC Development

AGESA PI 0.0.7.4 released Mar 12, 2026. Added FAST, UPP, UBPS power management features. OpenBMC v2.18.0.074 released Apr 1, 2026 with ALERT-L driver, APML 5.1.1 support.

PC
May 2026 · Upcoming

Production Candidate — Final Electrical Validation

128C / 256C (CCD32), 64C (CCD12) · Fused non-secure · AGESA PI 0.0.8.0 (Min PI for PC support). Final electrical margining at POR speeds, initial thermal testing.

PR
July 2026 · Target

Production Ready — Certification & Pilot

Up to 256C (CCD32), up to 96C (CCD12) · Fused secure · AGESA PI 0.0.9.0 (Production Ready, All Features Optimized). Final system validation, certification tests, pilot/production testing.

🚀
H2 2026 · Production Target

AMD Production Ramp (IP)

Fused secure production. AGESA PI 1.0.0.0 (Production Release). 7-year product availability planned. Target TDP up to 600W with full SP7 IRM Group support.
